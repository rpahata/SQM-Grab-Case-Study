# SQM-Grab-Case-Study
Software Quality Management Final Project – Grab Mobile Application

Software Quality Management Case Study

Grab Mobile Application

Project Overview

This project analyzes the software quality management practices of the Grab mobile application. The study focuses on quality attributes, defect analysis, CMMI maturity assessment, and risk management.

Quality Policy

The system should ensure:
	•	High reliability
	•	Strong security protection
	•	Fast performance
	•	Good usability
	•	Continuous service availability

Quality Objectives
	•	Reduce critical defects
	•	Improve testing coverage
	•	Strengthen risk management
	•	Standardize development processes

CMMI Assessment

Current maturity level: Level 2 – Managed
Improvement goal: Reach Level 3 – Defined
